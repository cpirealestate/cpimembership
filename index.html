Para Parceiros

Como validar os descontos dos membros

Como escanear o código

  1. Abra o aplicativo de validação CPI Membership
  2. Posicione a câmera sobre o código QR do cliente
  3. Aguarde a confirmação do sistema
  4. Aplique o desconto conforme indicado

Informações importantes

  • Verifique sempre a validade do cartão
  • Certifique-se de que o cliente é o titular do cartão
  • Em caso de problemas, entre em contato com o suporte
  • Descontos não são cumulativos com outras promoções
